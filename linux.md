###_file system_###

ls variations:

    ll
    ls -al
    ls -thor

`al` list hidden file, `ll` is short, (not supported by every version of linux), `thor` because of this is funny.

wc:

wc can be used on folders, list view and sum will be displayed:

    wc Folder/
